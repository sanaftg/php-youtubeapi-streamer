# php-youtubeapi-streamer
特定（複数）配信者の新着動画またはライブをいい感じに取得してJSONにして吐きます。
複数のチャンネルから配信状態引っ張ったりするために作りました。

自由に使ってOKですが、動作は保証しません。

※GoogleAPIライブラリが必要

通常の手法よりクォータが大幅に削減できます。
